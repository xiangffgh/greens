------------------常用操作----------------------
打开最近列表 Ctrl + E
查找类       Ctrl + N
替换         Ctrl + R
查找         Ctrl + F
关闭文件 Shift + Click
回到上一次编辑的地方 Ctrl + Shift + BackSpace

------------------优化代码-----------------------
格式化代码     Ctrl+Alt+L
移除没用的import  Ctrl + Alt + O
显示类的大纲      Ctrl + F12

------------------生成代码-----------------------
自动导入包,自动修正                    Alt + Enter
自动代码                           Ctrl + J

把代码包在一块内，例如try/catch    Ctrl+Alt+T
提取方法                           Ctrl + Alt + M
